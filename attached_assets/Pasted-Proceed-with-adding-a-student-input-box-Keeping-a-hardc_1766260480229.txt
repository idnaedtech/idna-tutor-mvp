# Goal

Let you type a student ID, then:

* **Start Session** uses that student ID
* **Resume Last Session** uses that student ID

---

# Step A: Add the input box (HTML)

Open `static/index.html`.

Find the area near your **Start Session** button. Add this just above it:

```html
<div style="margin: 8px 0;">
  <label for="studentIdInput">Student ID:</label>
  <input id="studentIdInput" type="text" value="s1" style="width: 320px;" />
</div>
```

Simple: you can type `s1`, `s2`, etc.

---

# Step B: Replace the hardcoded JS variable (JavaScript)

Find this line (you mentioned it):

```js
let studentId = "00000000-0000-0000-0000-000000000001";
```

Replace it with:

```js
function getStudentId() {
  const el = document.getElementById("studentIdInput");
  return (el?.value || "").trim() || "s1";
}
```

Now student id comes from the input box.

---

# Step C: Update Start Session to use typed student ID

In your `start()` (or whatever function runs on Start Session), replace any use of `studentId` with:

```js
const studentId = getStudentId();
```

# Step D: Add “Resume Last Session” button + function

## 1) Add button (HTML)

Place it near Start Session:

```html
<button onclick="resume()">Resume Last Session</button>
```

## 2) Add function (JS)

```js
async function resume() {
  const studentId = getStudentId();
  const res = await fetch(`/api/resume?student_id=${encodeURIComponent(studentId)}`);
  const data = await res.json();

  if (data.status !== "ok") {
    addLine(`Tutor: No previous session found for student ${studentId}.`);
    return;
  }

  // store session_id in the same variable you already use
  sessionId = data.session_id;

  addLine(`Tutor: Resumed session ${data.session_id} (topic ${data.topic_id}, state ${data.state}).`);
}
```

⚠️ Replace `addLine()` with your real function name (you already have something that prints “Tutor: ...”).
Also replace `sessionId` with your real global variable (you already display it at the top).
